# Simple-List-App

A simple app that fetches and displays a list of users from an API
Features:
- Download a list of user from api
- Save to local DB
- Click to remove user
- Refresh to redownload data

Technologies:
- Room
- Coroutines
- Retrofit
- Glide
- MVVM
- Repository pattern
